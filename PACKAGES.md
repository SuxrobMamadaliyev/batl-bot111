# Foydalanilgan Kutubxonalar

## Asosiy Kutubxonalar

### 1. Node.js Core Modullari
- `path` - Fayl yo'llarini boshqarish
- `fs` - Fayl tizimi bilan ishlash
- `http/https` - HTTP/HTTPS server yaratish

### 2. Tashqi Paketlar
- `node-telegram-bot-api` (^0.66.0) - Telegram Bot API bilan ishlash uchun
- `express` (^4.21.2) - Web server yaratish uchun
- `dotenv` (^16.6.1) - Environment variable'lar bilan ishlash

## Rivojlanish Dasturlari
- `nodemon` (^2.0.20) - Avtomatik qayta yuklash uchun

## Paket Boshqaruv
- `npm` - Node.js paket menejeri

## Versiya Boshqaruvi
- `git` - Versiya boshqaruvi tizimi

## Qo'shimcha Imkoniyatlar
- `cors` - Cross-Origin Resource Sharing sozlamalari
- `multer` - Fayl yuklash uchun

## Ishlatilgan API-lar
- Telegram Bot API - Botning asosiy funksionalligi uchun

## O'rnatish

Barcha kerakli paketlarni o'rnatish uchun:

```bash
npm install
```

## Rivojlatchilar Uchun

Rivojlatchilar uchun qo'shimcha paketlar:

```bash
npm install --save-dev nodemon
```
